# termux-drb
using DRb and your phone's termux to interop

on Mobile
```shell
ruby main.rb
```

on PC
```shell
MOBILE=druby://192.168.1.2:8787 ruby client.rb
```

## Install
on both side
```shell
git clone https://github.com/RGSS3/termux-drb
```


If you don't get a GIT on your phone, it's easy to 
```shell
apt install git
```
